Automated Data Mining and Preprocessing Pipeline

This page describes the data mining and preprocessing pipeline used to build the fish species dataset.

The dataset creation process was time-consuming and well-suited for automation. An automated pipeline was developed to mine image data, process images, and organize the dataset. This approach reduces manual effort while maintaining dataset quality. The pipeline is modular and easily customizable, allowing new data sources to be incorporated and image processing methods to be applied across the entire dataset.

  • iNaturalist — community-sourced wildlife observations with labeled images
  • GBIF (Global Biodiversity Information Facility) — aggregated biodiversity occurrence records with image data

Pipeline Overview

  1. Download images by species using automated scripts (iNaturalist + GBIF)
  2. Organize images into species-specific folders
  3. Manually filter images (keep / reject / maybe)
  4. Isolate high-quality images into a clean dataset
  5. Apply background removal and normalization
  6. Resize and standardize images for model training
  7. Assemble final dataset for machine learning pipeline
